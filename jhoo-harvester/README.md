# jhoo Harvester Chrome Extension

## Install
1. Open Chrome and go to chrome://extensions
2. Enable "Developer mode" (top right toggle)
3. Click "Load unpacked"
4. Select the jhoo-harvester folder

## Use
1. Go to LinkedIn Jobs search with your filters set
2. Click any job card to expand its description
3. The extension silently captures it — watch the green button count go up
4. Keep clicking jobs as normal
5. When done, click the green button to send everything to Supabase
6. The scorer will pick them up automatically
